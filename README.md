# 🌍 Global Economic & Social Indicators Dashboard

> **Tool:** Microsoft Power BI  
> **Data Source:** World Bank API / Global Indicators Dataset  
> **Coverage:** Multi-region, multi-year longitudinal analysis  
> **GitHub:** [View Repository](https://github.com/shashikantkchauhan/World_indicator_analysis)

---

## 📌 Project Overview

This Power BI project aggregates and visualizes global development data across 
multiple indicators to uncover socioeconomic trends, regional disparities, and 
technology adoption patterns worldwide.

The dashboard spans **3 interactive report pages**, each focusing on a different 
lens of global development analysis.

---

## 📊 Dashboard Pages

| Page | Focus Area |
|---|---|
| **Page 1** | GDP per Capita & Internet Penetration by Region & Year |
| **Page 2** | Youth Unemployment & Poverty Reduction Trends |
| **Page 3** | Immunization Rates & Social Development Indicators |

---

## 💡 Key Indicators Analyzed

- 📈 **GDP per Capita** — Economic output trends across regions
- 🌐 **Internet Penetration** — Technology adoption over time
- 👥 **Youth Unemployment** — Labour market analysis by country
- 💉 **Immunization for Kids** — Healthcare access indicators
- 🏚️ **Poverty Reduction** — Socioeconomic disparity tracking

---
